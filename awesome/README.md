# AwesomeWM

Awesomwe is a tiling window manager for the X windowing system configured in
lua.

## Dependencies

- [Victor Mono Nerd Font](https://github.com/ryanoasis/nerd-fonts) based on
  [Victor Mono](https://github.com/rubjo/victor-mono)
